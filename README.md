# Projeto Portfólio 👨‍💻

Bem-vindo ao repositório do meu **Projeto Portfólio**! Este projeto é o trabalho final desenvolvido durante o curso de HTML e CSS do **Curso em Vídeo**, marcando a consolidação dos conhecimentos adquiridos ao longo das aulas.

## 📝 Sobre o Projeto

Este portfólio foi criado para atuar como uma vitrine profissional e centralizar meus projetos, habilidades e informações de contato. O foco principal foi aplicar conceitos avançados de estruturação web, design responsivo e introduzir interatividade na página. Este portifólio irá crescer ao longo do tempo.

## 🛠️ Tecnologias Utilizadas

Este projeto marca uma evolução na stack, incluindo interatividade:

* **HTML5** - Estruturação semântica de todas as seções do portfólio.
* **CSS3** - Estilização completa, layout responsivo e animações visuais.
* **JavaScript** - Adicionado para criar interatividade e dinamismo na interface do usuário. Esse trecho foi feito pelo curso em vídeo.

## 📁 Estrutura de Arquivos

O projeto está organizado com uma arquitetura bem dividida:

```text
projeto-portfolio/
